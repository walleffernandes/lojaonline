🛍️ Loja Online

Projeto de e-commerce desenvolvido com foco em layout moderno, responsividade e experiência do usuário.

A aplicação simula uma loja virtual com listagem de produtos, carrinho e interface intuitiva.

🚀 Tecnologias Utilizadas

🌐 HTML5

🎨 CSS3

⚡ JavaScript

📱 Design Responsivo

🧩 Estrutura modular de arquivos

🎯 Funcionalidades

✅ Página inicial com produtos em destaque

✅ Layout responsivo (mobile-first)

✅ Estrutura preparada para carrinho de compras

✅ Organização semântica do HTML

✅ Interface simples e objetiva

🛠️ Como Executar o Projeto
🔹 Opção 1 – Abrir direto no navegador

Basta abrir o arquivo:
index.html

🔹 Opção 2 – Usar Live Server (Recomendado)

Se estiver usando VS Code:

Instale a extensão Live Server

Clique com botão direito em index.html

Selecione Open with Live Server
